Commands that work in every server:

.help : sends a list of all the commands.
.help (command name) : shows information on the mentioned commands.

.say (message) : repeats the content of the message.
.repeat (message) : same as say.

.addrole (member) (role name) : adds a role to a user (admin needed).
.removerole (member) (role name) : removes a role from a user (admin needed).
